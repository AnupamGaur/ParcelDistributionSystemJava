# Approach

## Objective
To create a flexible, rule-based parcel distribution system that categorizes parcels for processing by various departments.

---

## Tech stack

- Java
- Console for UI(React.js if time allows)


## Implementation 

- create POJOs to represnt entities in the system.
- parse the given xml using DOM.
- implement rules logic to distribute parcels ex - Parcels upto 1 kg directed to "Mail".
- create rules configuration system via a JSON file.
- implement complete app engine and console app.
- create unit tests for models, parser, rules, etc.
